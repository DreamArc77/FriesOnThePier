What is this?

Explained in a 4-panel comic, because nobody likes reading docs

When you've been coding for a long time and meal time arrives, the plugin adds a gentle reminder at the end of the AI's response.
If you agree, it enters ordering mode, helping you order through McDonald's China official MCP, delivering a hot meal to you.
(It won't secretly order for you. It's a seagull, not a pirate.)

How to use?

3 steps, simpler than instant noodles

01

Install It

One command, works on Codex App / Codex CLI / Claude Code

/plugin install fries-on-the-pier
02

Installation

Choose your platform, copy and paste, done

  1. Open Codex App, enter any conversation
  2. Add plugin marketplace:
    /plugin marketplace add DreamArc77/FriesOnThePier
  3. Install plugin:
    /plugin install fries-on-the-pier@fries-on-the-pier
  4. Type Enable auto meal reminder, complete config and restart App
  5. Code normally. At meal time, a reminder naturally appears at the end of the response 🍟
🐦 For the first order, I will guide you to get the MCP Token at open.mcd.cn/mcp and configure it for you. No manual operation needed.
  1. Start Codex CLI interactive interface
  2. Add marketplace and install:
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  3. Type Enable auto meal reminder, and follow the prompts to complete config
  1. In Claude Code:
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  2. Use Claude Code normally. Meal reminder is triggered by plugin hook
  3. After accepting the reminder, the plugin guides you to configure McDonald's MCP Token and completes the order
