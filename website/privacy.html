隐私政策

生效日期：2026年4月30日

上海惊喜人科技有限公司（"我们"）开发了去码头整点薯条（FriesOnThePier），一款免费开源的 AI 编码助手插件。本隐私政策说明我们收集哪些信息、如何使用以及您的权利。

1. 插件做了什么

去码头整点薯条是一款运行在 Codex / Claude Code 等 AI 编码工具中的插件。它在本地工具链中运行，不需要注册账号。核心功能是在饭点时在 AI 回答末尾添加一句提醒，并在您同意后通过麦当劳官方 MCP 服务协助下单。

2. 我们收集的信息

插件本身不收集任何个人数据：

  • 不向我们的服务器传输任何数据。
  • 不要求账号注册。
  • 所有配置（包括 MCP Token）仅存储在您设备本地的 Codex / Claude Code 用户配置文件中。
  • 不记录、上传或存储您在会话中输入的任何文字或操作。

3. 麦当劳 MCP Token

如果您选择使用点餐功能，需要提供麦当劳中国官方 MCP Token。该 Token：

  • 仅存储在您本地设备上的 Codex / Claude Code 用户级配置或环境变量中。
  • 仅用于代表您直接向麦当劳 MCP 服务发起请求。
  • 不会传输到或存储在我们的服务器上。

麦当劳 MCP 服务由麦当劳中国运营，其数据处理规则请参考麦当劳中国隐私政策。

4. 开源透明

去码头整点薯条在 MIT 许可证下完全开源。您可以在 github.com/DreamArc77/FriesOnThePier 查看所有代码，不存在隐藏的数据收集机制。

5. Cookie 与分析

本官网不使用广告 Cookie、第三方分析工具或追踪像素。

6. 您的权利

由于插件不收集个人数据，我们不持有任何可供访问或删除的个人信息。如有任何疑问，请通过下方联系方式联系我们，我们将在30天内回复。

7. 政策更新

我们可能不时更新本隐私政策。更新内容将在本页面发布，并注明新的生效日期。
