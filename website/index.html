它是啥玩意儿？

用四格漫画讲清楚，因为没人喜欢读文档

插件在你长时间写代码、刚好到了午饭或晚饭时间时，在回答末尾轻轻补一句提醒。
你如果同意，它进入点单模式，通过麦当劳中国官方 MCP 帮你完成点单，为你送上一顿热餐。
（它不会偷偷帮你下单，它是海鸥，不是海盗。）

怎么用？

三步，比泡面还简单

01

装上去

一行命令，Codex App / Codex CLI / Claude Code 通吃

/plugin install fries-on-the-pier
02

忘记它

正常写代码，插件在后台默默盯着时间，你不用操心

03

吃薯条

饭点到，AI 轻声提醒。说"帮我点"，海鸥出动，热餐送达

安装

选你的平台，复制粘贴，完事

  1. 打开 Codex App，进入任意对话
  2. 添加插件市场：
    /plugin marketplace add DreamArc77/FriesOnThePier
  3. 安装插件：
    /plugin install fries-on-the-pier@fries-on-the-pier
  4. 输入 启用自动饭点提醒，完成配置后重启 App
  5. 正常写代码。到饭点时，回答末尾会自然出现提醒 🍟
🐦 首次点餐时，我会引导你去 open.mcd.cn/mcp 获取 MCP Token，然后帮你配好，你不用手动操作。
  1. 启动 Codex CLI 交互界面
  2. 添加市场并安装：
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  3. 输入 启用自动饭点提醒，按提示配置完成
  1. 在 Claude Code 中：
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  2. 正常使用 Claude Code。饭点提醒由插件 hook 触发
  3. 接受提醒后，插件引导你配置麦当劳 MCP Token，继续完成点单
