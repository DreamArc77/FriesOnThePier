これは何ですか？

4コマ漫画で説明します、誰もドキュメントなんて読みたくないから

長時間コーディングしていて、ちょうどランチやディナーの時間になった時、プラグインがAIの回答の最後にそっとリマインダーを追加します。
あなたが同意すると、注文モードに入り、マクドナルド中国公式 MCP を通じて注文を完了し、温かい食事をお届けします。
(勝手に注文することはありません。カモメであり、海賊ではありません。)

使い方

3ステップ、カップラーメンより簡単

01

インストール

1つのコマンドで、Codex App / Codex CLI / Claude Code すべてに対応

/plugin install fries-on-the-pier
02

忘れる

普通にコーディングしてください。プラグインがバックグラウンドで静かに時間を見守っています。気にする必要はありません。

03

ポテトを食べる

食事の時間が来ると、AIが優しくリマインドします。「注文して」と言えば、カモメが出動し、温かい食事が届きます。

インストール方法

プラットフォームを選んで、コピー＆ペーストするだけ

  1. Codex App を開き、任意の会話に入ります
  2. プラグインマーケットプレイスを追加:
    /plugin marketplace add DreamArc77/FriesOnThePier
  3. プラグインをインストール:
    /plugin install fries-on-the-pier@fries-on-the-pier
  4. 自動食事リマインダーを有効にする と入力し、設定を完了してアプリを再起動します
  5. 普通にコーディングします。食事の時間になると、回答の最後に自然とリマインダーが表示されます 🍟
🐦 初めての注文の際、open.mcd.cn/mcp に案内して MCP Token を取得し、設定までお手伝いします。手動での操作は不要です。
  1. Codex CLI インタラクティブインターフェースを起動します
  2. マーケットプレイスを追加してインストール:
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  3. 自動食事リマインダーを有効にする と入力し、プロンプトに従って設定を完了します
  1. Claude Code の場合:
    /plugin marketplace add DreamArc77/FriesOnThePier
    /plugin install fries-on-the-pier@fries-on-the-pier
  2. Claude Code を通常通り使用します。食事リマインダーはプラグインのフックによってトリガーされます
  3. リマインダーを受け入れた後、プラグインがマクドナルド MCP Token の設定を案内し、注文を完了します

よくある質問 (FAQ)

少しバカバカしいけど、とても理にかなった質問たち

このプラグインは勝手に注文して、知らないうちにお金が引かれたりしませんか？
🧑‍💻
🐦
しません。僕はカモメであって海賊ではありません。注文する前には必ず聞きますし、何も言わなければそこで待機します。勝手なことはしません。
マクドナルドは嫌なんですが、変更できますか？
🧑‍💻
🐦
現在はマクドナルド中国公式 MCP のみをサポートしています。でも、もし他の公式 MCP を見つけたら、PR を歓迎します。検討してあげてもいいですよ。
Token は安全ですか？こっそり保存したりしませんか？
🧑‍💻
🐦
Token はローカルの Codex / Claude 設定ファイルに書き込まれるだけで、僕には関係ありません。そもそも見ていません（というか読めません）。あなたの Token、電話番号、詳細な配達先住所を保存することはありません。
お腹空いてないんですが、リマインダーをオフにできますか？
🧑‍💻
🐦
不要だと言われれば何も言いません。ただ、あなたのことが少し心配なだけです。😢
