プライバシーポリシー

発効日：2026年4月30日

上海惊喜人科技有限公司（「当社」）は、無料のオープンソースAIコーディングアシスタントプラグインである FriesOnThePier を開発しました。このプライバシーポリシーでは、当社が収集する情報、その使用方法、およびユーザーの権利について説明します。

1. プラグインの機能

FriesOnThePier は、Codex や Claude Code などのAIコーディングツール内で動作するプラグインです。ローカルのツールチェーン内で動作し、アカウント登録は必要ありません。主な機能は、食事の時間にAIの回答の最後にリマインダーを追加し、ユーザーの同意を得た上で、マクドナルド中国の公式MCPサービスを通じて注文をサポートすることです。

2. 収集する情報

プラグイン自体は個人データを収集しません：

  • 当社のサーバーにデータを送信することはありません。
  • アカウント登録を要求することはありません。
  • すべての設定（MCP Tokenを含む）は、デバイス上のローカルにある Codex / Claude Code ユーザー設定ファイルにのみ保存されます。
  • セッション中に入力したテキストや操作を記録、アップロード、または保存することはありません。

3. マクドナルド MCP Token

注文機能を使用することを選択した場合、マクドナルド中国公式 MCP Token を提供する必要があります。この Token は：

  • デバイス上の Codex / Claude Code のローカルユーザーレベル設定または環境変数にのみ保存されます。
  • ユーザーに代わってマクドナルドMCPサービスに直接リクエストを送信するためだけに使用されます。
  • 当社のサーバーに送信または保存されることはありません。

マクドナルドMCPサービスは、マクドナルド中国によって運営されています。データ処理のルールについては、マクドナルド中国のプライバシーポリシーを参照してください。

4. オープンソースの透明性

FriesOnThePier はMITライセンスの下で完全にオープンソース化されています。すべてのコードは github.com/DreamArc77/FriesOnThePier で確認できます。隠されたデータ収集メカニズムは存在しません。

5. Cookie と分析

この公式サイトは、広告Cookie、サードパーティの分析ツール、またはトラッキングピクセルを使用していません。

6. ユーザーの権利

プラグインは個人データを収集しないため、当社はアクセスまたは削除できる個人情報を保持していません。ご質問がある場合は、以下の連絡先までお問い合わせください。30日以内に返信いたします。

7. ポリシーの更新

当社は、このプライバシーポリシーを随時更新する場合があります。更新はこのページに新しい発効日とともに掲載されます。
